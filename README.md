## Quick Start

1. Clone this repository
  ```
  git clone https://github.com/AlterClassIO/material-dashboard-react
  ```
2. Install dependencies
  ```
  cd project-folder
  npm install
  ```
3. Start development server
  ```
  npm start
  ```
4. Enjoy!
